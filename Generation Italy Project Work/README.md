# Statistical correlation between market drivers and various stock characteristics
This is the final Project Work for the "Junior Data Engineer" DATITA031 course of Generation Italy  
The task was the following:

EN: "The research group of the Department of Economic Sciences aims to investigate how market drivers and various stock characteristics exhibit statistical correlations.
You are tasked with analysing the relationships between key variables to describe market patterns and to:
-determine whether correlations exist between stock volatility and market sector;
-assess whether stock trends are influenced by historical events."

What was done:  
- Data cleaning and analysis with python;
- upload datasets on MySQL server;
- server queried through RESTful API with Postman and HTML website;
- Power BI dashboard;
- Final dissertation on the project in italian
